# 🌳 SkillTree

**Визуальное построение и отслеживание прогресса освоения IT-навыков**

SkillTree — это веб-приложение для создания интерактивных деревьев навыков. Оно помогает IT-специалистам, студентам и преподавателям наглядно планировать обучение, отслеживать прогресс и контролировать дедлайны.

## ✨ Возможности

- 📊 **Визуальный редактор** — панорамирование, масштабирование, перетаскивание узлов
- 🌲 **Дерево навыков** — разделы и навыки с иерархической структурой
- ✅ **Статусы** — «Не усвоен», «В процессе», «Усвоен»
- ⏰ **Дедлайны** — цветовая индикация (зелёный / жёлтый / красный)
- 📈 **Прогресс** — автоматический расчёт прогресса разделов и всего дерева
- 📚 **Библиотека готовых деревьев** — шаблоны для быстрого старта
- 📊 **Статистика** — круговые диаграммы, календарь активности
- 📄 **PDF-резюме** — экспорт списка освоенных навыков
- 👑 **Админ-панель** — управление пользователями и библиотекой
